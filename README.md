# Crime-Management-System
A web-based Crime Management System that enables civilians and officers to report, manage, and monitor crime records efficiently.
Crime Management System is a full-stack web application designed to streamline the process of crime reporting and management.
It allows users to sign up either as a civilian or an officer. Civilians can report crimes online, while officers have additional privileges such as viewing and deleting crime reports.
The system maintains transparency through audit logs and a structured database design.

 Features
Secure Login and Signup system
Dual role-based access: Officer and Civilian
Crime Reporting functionality for all users
Crime Deletion and management privileges for officers
SQL Triggers for automated audit logging
Clean and responsive Bootstrap-based interface

 Technologies Used
Frontend: HTML, CSS, Bootstrap
Backend: Python (Flask)
Database: MySQL
Version Control: Git & GitHub

 Future Enhancements
Integration of real-time crime tracking using APIs
Adding OTP-based verification for user registration
Generating visual crime statistics and reports
